=======================================================================================
SYSTEM TYPES DATA FILE
=======================================================================================

System Physical Type:
  Normal, Nebulae, Black Hole

Background Bitmap:
  This is the bitmap which is displayed in the background of the 
  solar system window. There needs to be 1 of these in the 
  Pictures\Game\Screens\1024x768 and 1 in the 
  Pictures\Game\Screens\800x600 directories. The exact same name
  should be used. One is a larger image for larger displays (1024x768),
  the other is for smaller displays.

Empires Can Start In:
  Whether one of an empire's planets can start within a system of this 
  type when the game is created. 

Mask Background Objs:
  Whether items such as planets, warp points, etc. need to be masked
  when drawn on the background bitmap.

Non-Tiled Center Pic:
  Picture displayed should be shown in the center of the combat map and
  is not tiled. 

Abilities:
  System - Sight Obscuration, System - Sensor Interference, System - Shield Disruption,
  System - Damage, System - Ability Required

WP Stellar Abil Type:
  The name of a stellar ability type from the StellarAbilityTypes.txt file.
  This will be applied to all warp points generated for this system.

Number of System Objs:
  The number of objects that will be placed in this system. This does not
  include warp points.

Obj X Physical Type:
  Planet, Asteroids, Storm, Sun, Destroyed Star, Comet

Obj X Position:
  Ring 1 : Center Square of System.
  Ring 2 - 7 : 1 - 6 squares out from Center of System. 
    The actual position in the ring is completely random.
  Coord X, Y : A coordinate position within the system. 
    X is horizontal position from 0 to 12, 
    Y is vertical position from 0 to 12.
    There is no randomness to this placement.
  Same As X : The same position as Obj X. 
    There is no randomness to this placement.
  Circle Radius X: A true circle in the system at distance X. 
    The actual position in the ring is completely random.

Obj X Stellar Abil Type:
  The name of a stellar ability type from the StellarAbilityTypes.txt file.

Obj X Size:
  Any, Tiny, Small, Medium, Large, Huge

Obj X Atmosphere:
  Used only for planets.
  Any, None, Methane, Oxygen, Hydrogen, Carbon Dioxide

Obj X Composition:
  Used only for planets.
  Any, Rock, Ice, Gas Giant


Allowable Values for Star Age:
  Young, Average, Old, Ancient

Allowable Values for Star Color:
  Yellow, Red, Purple, Green, Blue, White, Orange

Allowable Values for Star Luminosity:
  Dim, Average, Bright, Super Bright
